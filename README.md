# Loan Policy RL Project

## Folder Structure
- 01_EDA/  
- 02_Supervised_DL_Model/  
- 03_Offline_RL/  
- final_report/  
- data/  (dataset goes here; not uploaded to GitHub)

## How to Work With This Repo
1. Download LendingClub dataset from Kaggle.
2. Put it in the `data/` folder locally (do not upload full data to GitHub).
3. Run notebooks in this order:
   - 01_EDA
   - 02_Supervised_DL_Model
   - 03_Offline_RL

## Purpose
Build:
- A supervised deep learning model to predict default  
- An offline RL agent to learn loan approval policy
